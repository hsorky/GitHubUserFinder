# GitHubUserFinder
Программа для поиска пользователь GitHub. Приложение создано в обучающихся целях для итоговой аттестации по курсу "Код будущего", Мокроусов Тимофей

Данная программа выводит никнеймы пользователь GitHub по ключевому слову. Так же программа может сохранять никнеймы на выбор в отдельный файл расширения JSON.
Интерфейс реализован при помощи стандартной библиотеки tkinker, в программе использована стандартна библиотека requests для отправления запроса на адрес "https://api.github.com/search/users".

**Требования для запуска**
- Python 3.8+
- Библиотека requests (устанавливается через pip)

**Для запуска программы:**
- скачайте файл GitHubUserFinder.py
- запустите файл через коммандную строку (cd путь_к_файлу) или через интегрированные среды разработки python.
  
Интерфейс приложения состоит из трёх кнопок ("Избранное", "Поиск" и "В избранное"), поля вывода списка listbox.
<img width="593" height="528" alt="image" src="https://github.com/user-attachments/assets/cae846b9-714c-459a-9e3a-6874f4a7f082" />
Чтобы найти пользователя необходимо ввести его никнейм в поиск и нажать на кнопку "Поиск", после чего в списке будет выведено 18 позиций никнеймов, похожих на введённый.
Для добавления никнейма пользователя в избранное необходимо выбрать любой никнейм из выведенного списка и нажать на кнопку "В избранное".
Для просмотра списка избранных необходимо нажать на кнопку "Избранное", после чего в поле вывода будет выведены все позиции списка избранных.
